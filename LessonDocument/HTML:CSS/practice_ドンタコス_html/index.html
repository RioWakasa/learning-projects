とうもろこし100%にこだわっているから、
その配合から見直し、とうもろこし本来の
甘さと香ばしさをより感じる生地へ。

試行錯誤を繰り返し、
絶妙な焼きと揚げ時間で生み出した、
従来よりパリッ、サクッ！な口溶けの良い
クリスピー食感。

生地の味わいも感じてほしいから、
しっかりとした旨みと、素材の味わいを感じる
食べ飽きない一枚に仕上げました。

ドンタコスは、とうもろこしの粒を丸ごとそのまま石臼で挽いた生地を薄くのばし、オーブンで温度を変えながら焼き上げ、高温の油でカラッと揚げています。

HISTORY
ドンタコスの歴史

1994年の発売以来ずっと、「ドンタコス」は日本生まれの
トルティアチップスとして、皆様に愛され続ける、
湖池屋を代表するロングセラーブランドの一つです。

ドンタコスCM「ストリート・オブ・ドンタコス」篇

1994年「ドンタコス」発売当初のTVCM「ストリート・オブ・ドンタコス」篇は、本場のメキシカンな世界感にこだわり、湖池屋初の海外で撮影されたものです。そのリズミカルなテンポは、思わず口ずさみたくなり、当時話題になりました。
